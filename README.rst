===================
lcheapo_obspy
===================

The missing link between LCHEAPO data and ObsPy

Overview
======================

Command-line programs:
----------------------

- lcplot: plot an LCHEAPO file
- lctest: plot LCHEAPO tests
- lc2ms_w: converts LCHEAPO file to miniseed (basic, NO drift correction)
